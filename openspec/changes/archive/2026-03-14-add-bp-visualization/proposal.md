# Change: 构建BP反向传播算法交互式可视化网站

## Why
BP（反向传播）算法是深度学习的基石，但其数学推导过程抽象，初学者难以直观理解前向传播、梯度计算和权重更新的全过程。通过构建一个交互式可视化网站，用户可以逐步观察数据在神经网络中的流动、损失的计算、梯度的回传以及权重的更新，从而显著降低学习门槛。

## Reference
参考文章：https://zhuanlan.zhihu.com/p/40378224（一文搞懂反向传播算法）

该文章使用经典的 Matt Mazur BP教程数值案例，通过一个 2-2-1 的三层神经网络，以具体数值逐步演示 BP 算法全过程。本项目将该案例作为**默认演示场景**：

- **网络结构**：3层（2输入-2隐藏-2输出），Sigmoid 激活函数
- **初始参数**：
  - 输入：i1=0.05, i2=0.10
  - 目标输出：o1=0.01, o2=0.99
  - 权重：w1=0.15, w2=0.20, w3=0.25, w4=0.30, w5=0.40, w6=0.45, w7=0.50, w8=0.55
  - 偏置：b1=0.35, b2=0.60
  - 学习率：η=0.5
- **前向传播**：net_h1=0.3775 → out_h1=0.5933 → net_o1=1.1059 → out_o1=0.7514
- **损失计算**：E_total=0.2984（MSE）
- **反向传播**：链式法则逐层求梯度 δ_o1=0.1385, δ_h1=0.0121
- **权重更新**：w5: 0.40→0.3589, w1: 0.15→0.1497
- **收敛验证**：10000次迭代后误差降至 0.000035

## What Changes
- 搭建基于 Vite 的前端工程，采用纯 HTML/CSS/JS 实现
- 实现神经网络数学模型（支持多层前馈网络，Sigmoid/ReLU 激活函数）
- 实现 Canvas 神经网络结构可视化渲染引擎（节点、连线、权重标注）
- 以参考文章的经典数值案例（2-2-2网络）作为默认演示场景
- 实现前向传播动画（数据逐层流动，节点激活高亮，显示 net/out 值）
- 实现损失计算展示（实际输出 vs 期望输出，MSE 损失值实时显示）
- 实现反向传播动画（梯度逐层回传，链式法则可视化，连线颜色/粗细表示梯度大小）
- 实现权重更新可视化（更新前后对比，学习率影响展示）
- 实现交互控制面板（网络结构配置、学习率调节、单步/自动训练、速度控制）
- 实现 BP 算法原理介绍页面（含 KaTeX 数学公式渲染、与参考文章一致的推导过程）
- 实现训练损失曲线图表

## Impact
- Affected specs: bp-algorithm-visualization, neural-network-interactive-controls, bp-algorithm-education
- Affected code: 全新项目，从零搭建
- 用户影响: 提供完整的 BP 算法学习和可视化体验，默认场景与参考文章数值一致，便于对照学习
