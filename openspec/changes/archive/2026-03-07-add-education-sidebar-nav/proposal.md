# Change: 为算法原理页面添加侧边栏目录导航

## Why
当前算法原理页面内容较长（包含网络结构、前向传播、损失计算、反向传播、权重更新、关键概念等多个章节），用户需要不断滚动才能找到目标内容，缺乏快速定位能力。参考常见的技术文档页面布局（如左侧固定目录 + 右侧内容区），添加侧边栏导航可以显著提升浏览体验。

## What Changes
- **新增左侧侧边栏导航**：在算法原理页面左侧添加固定定位的目录导航栏，列出所有章节标题（h3 级别）
- **点击跳转**：点击导航项平滑滚动到对应章节
- **当前位置高亮**：滚动时自动高亮当前可见章节对应的导航项
- **响应式适配**：侧边栏在窄屏下可折叠或隐藏

## Impact
- Affected specs: `bp-algorithm-education`（修改）
- Affected code:
  - `src/pages/education.js` — 生成导航目录结构、绑定滚动监听和点击跳转逻辑
  - `src/styles/main.css` — 添加侧边栏布局样式、高亮状态、滚动同步样式
  - `index.html` — 可能需要调整教育页面容器结构
